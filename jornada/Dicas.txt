-----------Criando um projeto em Django e Python----------------
1- Crie um ambiente de trabalho: 
	- mkvirtualenv nome_do_ambiente_virtual

2- Instale o django dentro desse ambiente virtual:
	pip install django

3 - Entre dentro do ambiente criado:
	workon = ver os ambientes criados. 
	workon nome_do_ambiente = entrar no ambiente criado

4-Entre na pasta django
	cd django

5- Crie o projeto e de start para atribuir as configurações essênciais:
	django-admin startproject nome_do_projeto

6- Inicie o servidor
	dentro da pasta do projeto use o comando = python manage.py runserver

------------------------- Criando Apps no projeto ------------------------
1- Criar apps:
	python manage.py startapp nome_do_app (app seria a página do site)

2- Depois declare na página settings.py dentro do programa, os apps criados.

---------- dicas: 
pip list - ver o que foi instalado
activate = ativar o ambiente virtual
deactivate = desativar o ambiente virtual
